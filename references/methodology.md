# Методология хай-концепта

> Синтез четырёх школ: Анатолий Сульянов (русскоязычная брендинговая школа), Donald Miller (StoryBrand), April Dunford (Obviously Awesome), Chip & Dan Heath (Made to Stick), плюс Hollywood high-concept (Justin Wyatt).

## Что такое хай-концепт

**Хай-концепт** — смысловая конструкция в **одной фразе**, которая мгновенно отвечает на вопрос *«Что это за продукт и почему он мне нужен?»*, выводит бизнес из прямой конкуренции по цене/условиям и становится **позвоночником всех коммуникаций** (позиционирование, копирайт, дизайн, питч, реклама).

### Ключевые характеристики

- **Одна фраза** — пересказывается за 10 секунд, помещается в одно предложение
- **Конкретная боль + понятный результат** — не абстракция
- **Меняет критерий сравнения**, а не соревнуется в нём
- **Заставляет конкурентов оправдываться**
- **Провоцирует пересказ** — клиент сам делится
- **Позвоночник бизнеса** — пронизывает контент, PR, продукт, упаковку

### Что хай-концепт НЕ ЕСТЬ

- ❌ Логотип / визуальный стиль / упаковка
- ❌ Рекламная кампания / слоган на квартал
- ❌ Описание процесса («Мы оказываем...»)
- ❌ Список достоинств («качество, сервис, забота»)
- ❌ Абстракция без конкретного результата
- ❌ «Хайп без содержания»

## Формулы

### Сульянов

```
Сила концепта = скорость_понимания × сила_идеи
```

Если первое требует пояснений — концепт слабый. Если второе — банальное, концепт слабый.

### StoryBrand (Donald Miller)

```
"We help [X] do [Y] so they can [Z]."
```

- X — сегмент ЦА
- Y — конкретная задача / трансформация
- Z — глубинный результат (то, ради чего на самом деле)

Примеры:
- «We help [women over 40] [lose weight without diet] so they can [feel sexy again]»
- «We help [B2B SaaS founders] [build positioning] so they can [stop competing on price]»

### Hollywood high-concept (Justin Wyatt)

```
"When [inciting incident], [protagonist] must [goal] or else [stakes]."
Или сжатее: "X meets Y"  (Alien = "Jaws in space")
```

Используется когда нужно концепт-в-одну-строку через известную аналогию.

### Value Proposition Canvas (Osterwalder)

Три кубика:

- **Jobs** — что клиент пытается сделать (functional + emotional + social)
- **Pains** — что мешает / бесит / пугает
- **Gains** — что хочет получить (выгоды, эмоции, статус)

Концепт = пересечение product-side (pain relievers + gain creators) с customer-side (pains + gains).

## Три опоры создания (Сульянов + синтез)

### Опора 1. Слушай клиента

Самые сильные формулировки скрыты в **речи аудитории**. Ищи в эмоциях:

- «устал от...»
- «не понимаю...»
- «хочу, чтобы было просто...»
- «меня бесит что...»
- «пугает когда...»

**Концепцию нельзя выдумать в кабинете — её нужно «добыть»** через интервью и наблюдение.

Связка с другими школами:
- **Value Proposition Canvas**: Jobs / Pains / Gains
- **Made to Stick** (Heath): Emotional — адресуй конкретного человека, не статистику

### Опора 2. Продавай результат и новое состояние

Не процесс, не фичу — **как меняется жизнь клиента**.

- ❌ «Я провожу тренировки 3 раза в неделю»
- ✅ «За 90 дней клиент перестаёт стесняться отражения в зеркале»

Связка:
- **StoryBrand**: character → problem → guide → plan → action → **success/failure**. Трансформация — ядро.
- **Value Prop Canvas**: Gains, не Features.

### Опора 3. Сужай фокус

Хай-концепт = **чёткий выбор + осознанный отказ от всего лишнего**.

- Чем уже концепция, тем сильнее.
- «Маркетинг для стоматологий с оборотом от 10 млн» > «Маркетинг»
- «Законно списываем долги» > «Юридические услуги»
- «CRM для автосервисов 5-20 мест без абонплаты» > «CRM для бизнеса»

Связка:
- **Positioning** (Ries/Trout): *Own a word* — одно слово в голове клиента (Volvo = safety, FedEx = overnight, Stripe = developer-friendly payments)
- **Obviously Awesome** (Dunford): выбрать узкую рыночную категорию, где уникальные атрибуты становятся obvious

## Эффекты сильного концепта (Сульянов)

Когда концепт работает — измеряется в 5 эффектах:

1. **Концепт обесценивает альтернативы** — стандартное предложение конкурентов начинает звучать «как обычно», без отдельной агрессии в их сторону.
2. **Концепт заставляет конкурентов оправдываться** — им приходится объяснять, почему у них «не так».
3. **Концепт меняет критерий сравнения** — сравнивают уже не по цене / срокам, а по новой оси, которую задал ты.
4. **Концепт провоцирует UGC и пересказ** — люди сами рассказывают друг другу про продукт, делятся фразой.
5. **Концепт становится позвоночником коммуникаций** — из одной строки выводится сайт целиком, питч, посты, реклама, дизайн.

Если у вашего концепта **0 эффектов** — это, скорее всего, не концепт, а описание услуги. Если **1-2** — слабый концепт. **3+** — сильный.

## Когда концепт нужен и когда нет

### Нужен

- Запуск нового продукта/услуги/бренда — концепт = старт коммуникаций.
- Repositioning после провала старого позиционирования.
- Выход в новый сегмент / новую категорию.
- Конкуренция упирается в цену — концепт переключает ось сравнения.
- Сайт/презентация/питч плохо конвертируют — часто проблема в отсутствии концепта.

### Не нужен / нет смысла

- Commodity-бизнес с ценовой моделью и без амбиций строить бренд (например, оптовая дистрибуция базового сырья).
- Внутренний инструмент компании, который не продаётся снаружи.
- Краткосрочная промо-кампания — там нужен слоган, не концепт.
- Личный hobby-проект без коммерческих целей.
